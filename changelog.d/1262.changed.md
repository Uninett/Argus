Upgrade to Tailwind CSS v4 and DaisyUI v5.

NOTES:
Custom themes defined in `DAISYUI_THEMES` must now use
CSS variable syntax with explicit `--color-` prefixes. For example:

```python
DAISYUI_THEMES = [
    "dark",
    "light",
    {
        "mytheme": {
            "color-scheme": "dark",
            "--color-primary": "#006d91",
            "--color-secondary": "#f3b61f",
            # ... other color variables
        }
    },
]
```

See the updated documentation for full details on theme customization
